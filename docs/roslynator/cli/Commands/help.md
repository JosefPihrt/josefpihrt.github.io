# `roslynator help`

Displays help\.

## Synopsis

```
roslynator help <COMMAND>
-h, --help
-m, --manual
-v, --verbosity <LEVEL>
```

## Arguments

**`<COMMAND>`**

Command name\.

## Options

##### `-h, --help`

Show command line help\.

##### `-m, --manual`

Display full manual\.

##### `-v, --verbosity <LEVEL>`

Verbosity of the log\. Allowed values are q\[uiet\], m\[inimal\], n\[ormal\], d\[etailed\] and diag\[nostic\]\.
